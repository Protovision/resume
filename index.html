  • bitstuff
    https://github.com/Protovision/bitstuff
    C++ routines for type casting based on reinterpreting the underlying bits

    C++ has a relatively new standard function std::bit_cast which performs a conversion by reinterpreting object representation that would otherwise be illegal. It only works for types of equal object size, I have made variations of this function that works for sizes of differing types along with each operation being constexpr, enabling them to be subject to compile-time optimizations.

  • mht
    https://github.com/Protovision/mht
    A simple hashtable implementation in C.

    Out of both an interest and a need, I developed a hashtable implementation for C. It's supported operations include: inserting an element as a key/value pair, lookup of an element from it's key, deleting an element, deleting all elements, reallocating the hashtable's capacity and performing a rehash of all elements, and the ability to traverse all elements of the unordered hashtable. The implementation uses the open addressing hash collision resolution technique. Non-empty buckets are linked so that traversal is always strictly linear with respect to the number of elements. Pointers to table entries remain valid after table modification (unless a reallocation occurs) and the hashtable automatically grows in proportion to a load factor.

  • moonbase
    https://github.com/Protovision/moonbase
    A 2D graphical platform for executing embedded Lua scripts

    moonbase is able to execute zip files that contain Lua scripts, image files, and sound files. The Lua scripts contain the logic code that drives what the moonbase engine does whose capabilities include drawing 2D graphics, rendering text, rendering images, rendering sound, and restricted file system access. The application uses Lua coroutines to continually run the rendering loop, with callbacks into Lua on events such as mouse or keyboard input, or to pause the rendering loop to give time for the Lua script to continue it's execution of the main logic. The engine was written in C and uses SDL 2.
